# 🚀 C# Problem Solving

A collection of fundamental C# programming problems solved from scratch — great for learning the basics of C# syntax, data types, and logic.

## 📌 Notes

- All exercise classes expose a public static method (e.g., `Declare()`, `Swap()`) so they can be called from `Program.cs` without conflicts.
- Only `Program.cs` contains the `Main` entry point — this avoids the `CS0017` multiple entry point error.
